CSS(Cascading Style Sheets)

      마크업 언어(HTML)가 실제로 화면에 표시되는 방법을 기술하는 언어
      (모양, 색상, 크기, 배경이미지, 움직임, ...)

      하나의 요소에 여러 CSS 코드를 겹쳐서 원하는 스타일을 만들어냄

      W3C에서 CSS를 웹 표준으로 지정함

CSS 선택자(CSS Selector)

      현재 문서 내에서 특정 요소를 선택하는 문자 또는 문자열(기호)

      * 요소(Element) : 시작태그 + 속성 + 내용 + 종료태그
    ** CSS 코드를 작성하는 방법 **
  • head 태그에 style 태그를 작성한 후 내부에 css 코드 작성
  • 원하는 요소에 직접 style 속성을 이용해서 css코드 작성
  • 외부에 .css 파일을 생성하여 연결하는 방법

태그 선택자

      - 현재 문서 내에 특정 태그를 모두 찾아 선택하는 선택자

      [작성법]
      
      
        태그명 { css 코드; }

id(#) 선택자

        HTML 문서 내에서
        id가 일치하는 요소를 선택하는 선택자

        [작성법]
        
          #id속성값 { CSS 코드;}
  1. id 선택자 테스트 1
  2. id 선택자 테스트 2
  3. id 선택자 테스트 3

class(.) 선택자

        HTML 문서 내에서 class 속성 값이 일치하는 모든 요소를 선택하는 선택자

        [작성법]
        
          .class속성값{ css 코드;}
        

        *** 요소에 CSS를 적용하기 위한 선택자를 작성할 때
          id 선택자보다 class 선택자를 이용하는 것이 좋다!! **

          (왜? id 속성 자체가 CSS 보단 JS에 많이 사용되기 때문)

      단독 사용 시 : 문서에 존재하는 모든 요소 선택자

      다른 선택자와 혼용 사용 시 : 선택된 요소의 모든 자식/후손/형제
